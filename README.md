This repository contains Python-based web scraping projects using the **Beautiful Soup** library. These projects demonstrate how to extract structured data from different web environments, ranging from static practice sites to dynamic job portals.

## üìÇ Repository Contents

* **`web_scraping_fake_python.ipynb`**: A project focused on scraping the [Fake Python Jobs](https://realpython.github.io/fake-jobs/) site. This covers extracting job titles, company names, and locations from static HTML.
* **`web_scraping_wuzzaf.ipynb`**: A practical scraper for [Wuzzuf](https://wuzzuf.net/), a leading job portal in Egypt. It extracts real-time job listings, including titles, requirements, and dates posted.
* **`Wuzzaf.csv`**: The output data extracted from the Wuzzuf scraper.
